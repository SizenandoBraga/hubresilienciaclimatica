Importar e Exportar dados do Firebase

Ferramenta administrativa para migrar dados do Firestore entre projetos Firebase. Exporte de uma conta, baixe o JSON e importe na outra.

1. Configuração Firebase

Use a configuração do projeto de origem para exportar. Depois atualize para a configuração do projeto destino para importar.

2. Coleções

Informe uma coleção por linha. Os IDs dos documentos serão preservados.

3. Exportar dados

Depois de conectar no Firebase de origem, clique em exportar. Será baixado um arquivo JSON com os documentos.

4. Importar dados

Atenção: a importação usa o mesmo ID dos documentos. Se já existir documento com o mesmo ID no projeto destino, ele poderá ser atualizado.
